## Environment Prepare

Install `node_modules`:

```bash
yarn
```

### Start project

#### for app.parami.io
```bash
yarn start
```
#### for staging.parami.io
```bash
yarn dev
```

### Build project

#### for app.parami.io
```bash
yarn build
```
#### for staging.parami.io
```bash
yarn build:dev
```


You may want to put `127.0.0.1 app.parami.io` or `127.0.0.1 staging.parami.io` into your `HOSTS` when you are coding.
